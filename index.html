GtsVPN

GtsVPN es un fork de LibreVPN pero para crear redes del rango 10.26.0.0/16. Está pensado para poder integrar maquinas dentro de subredes y así poder agilizar el trabajo, teniendo accesibles en una red virtual a todas ellas sin importar su ubicación geografica o física.

Básicamente es una serie de scripts para configurar Tinc de modo fácil, cada nodo tiene un par de llaves RSA y los que manejan la red tendrán el poder de decidir que máquina entra a la red y cual no.

Descentralizada

Al utilizar Tinc, no se necesita de un server o nodo central, se pueden hacer todos los enlaces que sean necesarios sin necesidad de un servidor central.

Segura

Además de que todo el tráfico viaja encriptado por ser una VPN, al usar subrangos de 10.26.x.x se pueden configurar firewalls y servicios para atender solo a cierta parte de la red por ejemplo.

Libre y abierta

Al ser un fork de un proyecto libre, tanto el código como las discuciones se mantienen libres y abiertas para que cualquiera pueda participar del proyecto, para enviar errores/ideas, leer documentación y demás, se puede usar la página del proyecto en el tracker de GentiSoft:
